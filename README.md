# ExpensePro - Premium Financial Dashboard

A breathtaking, modern financial tracking application built with React and Vite. Focused on high-end glassmorphism design, responsive layouts, and a "Witty CFO" user experience.

## Features

- **Personal CFO Dashboard**: Centralized view of your total balance, income, and expenses.
- **Transaction Flow**: Clean, colored iconography to track daily spending activity.
- **Glassmorphism Design System**: Clean, readable, and modern interface.
- **Theme Switcher**: Complete Dark/Light Mode using precise CSS variables.
- **Responsive Architecture**: Works fluidly on both Desktop (Top Nav) and Mobile screens (Bottom Nav).

## Project Structure

- `src/components/` - Home for reusable parts (`Navbar`, `TransactionCard`).
- `src/pages/` - Top level views (`Dashboard`, `Login`).
- `src/index.css` - Centralized CSS variables, globals, and design tokens.
- `src/assets/` - Placeholder for future static images and icons.
- `src/utils/` - Utility functions.

## Getting Started

1. Ensure Node.js is installed.
2. Open your terminal in the root folder (`webapp`).
3. Install dependencies and run:
   ```bash
   npm install
   npm run dev
   ```

Enjoy tracking your financial mastery with ExpensePro!
