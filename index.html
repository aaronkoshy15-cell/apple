Fusion Architecture

Two dies. One monumental leap.

The state-of-the-art Fusion Architecture connects two third-generation 3-nanometer dies into a single SoC using advanced packaging. This integrates a powerful CPU, scalable GPU, Media Engine, unified memory controller, Neural Engine, and Thunderbolt 5 capabilities into one incredibly efficient powerhouse.

The World's Fastest CPU Cores

Featuring a new 18-core CPU architecture. 6 industry-leading super cores deliver the world’s fastest single-threaded performance, alongside 12 all-new performance cores optimized for power-efficient, multithreaded workloads.
